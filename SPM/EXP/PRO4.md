## ‚öôÔ∏è **Step 2: Earned Value Analysis (EVA)**

We‚Äôll calculate the three key values:

### **1Ô∏è‚É£ PV ‚Äì Planned Value**

‚Üí Budgeted cost of work *scheduled* till date.

### **2Ô∏è‚É£ EV ‚Äì Earned Value**

‚Üí Budgeted cost of work *performed* till date.
Formula:

> EV = (Planned Cost √ó % Completed)

### **3Ô∏è‚É£ AC ‚Äì Actual Cost**

‚Üí Actual cost incurred till date.

---

### üßÆ **Calculations**

| Task      | Planned Cost (PV) | % Completed | EV (Earned Value) | Actual Cost (AC) |
| --------- | ----------------- | ----------- | ----------------- | ---------------- |
| T1        | 10,000            | 100%        | 10,000            | 10,000           |
| T2        | 15,000            | 100%        | 15,000            | 17,000           |
| T3        | 30,000            | 80%         | 24,000            | 35,000           |
| T4        | 20,000            | 70%         | 14,000            | 23,000           |
| T5        | 15,000            | 30%         | 4,500             | 5,000            |
| **Total** | **90,000**        |             | **67,500 (EV)**   | **90,000 (AC)**  |

---

### **Formulas:**

[
CPI = \frac{EV}{AC}
]

[
SPI = \frac{EV}{PV}
]

---

### **Calculations:**

* **CPI = 67,500 / 90,000 = 0.75**
* **SPI = 67,500 / 90,000 = 0.75**

---

## üìà **Interpretation**

| Index          | Value | Meaning                                           |
| -------------- | ----- | ------------------------------------------------- |
| **CPI = 0.75** | < 1   | Project is **over budget** (cost overrun)         |
| **SPI = 0.75** | < 1   | Project is **behind schedule** (delayed progress) |

‚úÖ If CPI > 1 ‚Üí under budget
‚úÖ If SPI > 1 ‚Üí ahead of schedule

## üìò **Key Viva Definitions**

| Term                                 | Definition                                                        |
| ------------------------------------ | ----------------------------------------------------------------- |
| **Gantt Chart**                      | A visual project schedule showing tasks, timelines, and progress. |
| **Earned Value (EV)**                | The value of work actually completed expressed in budgeted terms. |
| **Planned Value (PV)**               | The planned or scheduled cost up to a specific time.              |
| **Actual Cost (AC)**                 | The actual expenditure on the project up to a specific time.      |
| **CPI (Cost Performance Index)**     | Efficiency of cost usage. ( CPI = EV/AC )                         |
| **SPI (Schedule Performance Index)** | Efficiency of schedule. ( SPI = EV/PV )                           |
| **CPI < 1**                          | Cost overrun.                                                     |
| **SPI < 1**                          | Project delay.                                                    |
